# Bibliography — CS 5463/4953 Term Project
## Survey on Parameter Server Frameworks for Distributed Machine Learning

This bibliography lists foundational, journal, and conference papers relevant to parameter server architectures, distributed gradient methods, consistency models, and large-scale ML systems.
